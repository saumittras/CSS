Position: static;

An element with position: static; is not positioned in any special awy; it is always positioned according to the normal flow of the page:

This div element has position: static;

position: relative;

An element with position: relative; is positioned relative to its normal position:

This div element has position: relative;

position:fixed;

An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled:

This div element has position fixed;

position:absulute;

An element with position:absulute; is positioned relative to the nearest positioned ancestor(instead of positioned relative to the viewport, like fixed):

This div element has position: relative;
This div element has position: absulute;
